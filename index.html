    Student mangement system using binary search tree

    A student management system using Binary Search Trees (BST) organizes student information like names, IDs, and grades. Each student is like a node in a tree, with smaller grades on the left and larger grades on the right. This helps quickly find students with specific grades or IDs. Adding new students or updating information is efficient because the tree structure keeps everything sorted. Additionally, BSTs can be used to perform operations like finding the highest or lowest grade in the class.
